Python 3 Reference

Python 3 quick reference for Pre-IG Stage 1.

Variables & Data Types

Assignment:

age = 13
price = 9.99
name = "Alice"
is_active = True

Data types:

intWhole numbers: 5, -3
floatDecimals: 3.14
strText: "hello"
boolTrue or False

Use snake_case for variable names: my_age, first_name

Input & Output 
print("Hello, World!")
name = input("Enter your name: ")
print("Hello,", name)

f-strings (format strings):

score = 95
print(f"Your score is {score}")

Concatenation:

greeting = "Hello, " + name + "!"
Type Casting 
age = int(input("Enter age: "))
price = float(input("Price: "))
text = str(42)   # "42"
count = int(3.7) # 3 (truncates)

Always cast input() to the correct type before arithmetic.

Comments 
# This is a single-line comment
x = 5  # inline comment
Strings 
name = "Alice"
print(len(name))          # 5
print(name.upper())       # "ALICE"
print(name.lower())       # "alice"
print(name[0])            # "A"
print(name[-1])           # "e"
print(name[1:3])          # "li"  (slice)

# Check contents
print("li" in name)       # True
print(name.startswith("Al"))  # True
print(name.endswith("ce"))    # True

# Check if a character belongs to a set of characters
grade = "B"
if grade in "ABC":
    print("pass")         # pass

# Check character type
print("a".isdigit())      # False
print("3".isdigit())      # True
print("A".isupper())      # True
print("a".islower())      # True

# Modify
print(name.replace("Alice", "Bob"))  # "Bob"
print("  hello  ".strip())           # "hello"
print("a,b,c".split(","))            # ['a', 'b', 'c']
print("42".rjust(6))                 # "    42"  (right-align in 6-wide field)
print("42".ljust(6))                 # "42    "  (left-align in 6-wide field)
print("42".rjust(6, "0"))            # "000042"  (fill with 0 instead of space)

# f-strings
age = 12
print(f"My name is {name}, I am {age}.")

# Concatenation
greeting = "Hello, " + name
print(greeting)           # "Hello, Alice"

Strings are immutable — methods return a new string, they do not change the original.

Operators

Arithmetic:

+Addition
-Subtraction
*Multiplication
/Division (float result)
//Integer (floor) division
%Modulus (remainder)
**Exponent / power

Comparison:

==Equal to
!=Not equal to
<Less than
>Greater than
<=Less than or equal
>=Greater than or equal

Logical:

andBoth conditions true
orEither condition true
notNegates condition

Assignment uses =. Equality test uses ==.

Selection — if / elif / else 
if score >= 90:
    print("A grade")
elif score >= 70:
    print("B grade")
else:
    print("Try again")

Always use 4 spaces for indentation. The colon : is required.

Iteration — for loop 
for i in range(5):
    print(i)       # 0 1 2 3 4

for i in range(1, 6):
    print(i)       # 1 2 3 4 5

for i in range(0, 10, 2):
    print(i)       # 0 2 4 6 8

Use for when the number of iterations is known.

break — exit the loop early:

for i in range(10):
    if i == 5:
        break
    print(i)       # 0 1 2 3 4

break stops the loop immediately. Any code after the loop still runs.

Iteration — for each loop 
# Loop over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)   # apple, banana, cherry

# Loop over a string (character by character)
word = "hello"
for letter in word:
    print(letter)  # h, e, l, l, o

# Loop with index using enumerate()
for i, fruit in enumerate(fruits):
    print(i, fruit)  # 0 apple, 1 banana, 2 cherry

Use a for each loop to go through every item in a list or string without needing a counter.

Iteration — while loop 
count = 1
while count <= 5:
    print(count)
    count += 1

Use while when iterations depend on a condition.

Lists 
# Create
fruits = []               # empty list
colours = ["red", "blue"]

# Access (0-indexed)
print(colours[0])         # "red"
print(colours[-1])        # last: "blue"

# Modify
fruits.append("apple")
fruits.remove("apple")
del fruits[0]
fruits.reverse()          # reverses in place: [3,2,1]
fruits.sort()             # sorts in place: [1,2,3]

# Loop
for fruit in fruits:
    print(fruit)

# Length
print(len(fruits))
Functions

Void (procedure):

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")

With return value:

def add(a, b):
    return a + b

result = add(3, 4)
print(result)  # 7

No parameters:

def print_header():
    print("=" * 20)
    print("CGA PyLab")
    print("=" * 20)

Function names use snake_case. Call before using if defined below.

Turtle Graphics 
import turtle

t = turtle.Turtle()
t.forward(100)
t.right(90)
t.forward(100)
t.color("blue")
t.pensize(3)

# Draw a square
for i in range(4):
    t.forward(100)
    t.right(90)
t.forward(n)Move forward n pixels
t.backward(n)Move backward n pixels
t.right(angle)Turn right by angle°
t.left(angle)Turn left by angle°
t.penup()Lift pen (no drawing)
t.pendown()Put pen down
t.color("red")Set pen colour
t.fillcolor("blue")Set fill colour
t.begin_fill()Start filled shape
t.end_fill()End filled shape
t.circle(r)Draw circle radius r
t.goto(x, y)Move to position
t.speed(n)Speed 0=fast, 1=slow
Importing Libraries — math 
import math

print(math.pi)        # 3.141592653589793
print(math.sqrt(16))  # 4.0
print(math.floor(3.7)) # 3
print(math.ceil(3.2))  # 4

Common math functions:

math.piπ ≈ 3.14159…
math.sqrt(x)Square root of x
math.floor(x)Round down to nearest integer
math.ceil(x)Round up to nearest integer
math.pow(x, y)x to the power y (returns float)
math.log(x)Natural logarithm of x

Use import math at the top of your program. Access functions with math. prefix.
