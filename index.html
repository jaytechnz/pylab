Python 3 Reference

Python 3 quick reference for Pre-IG Stage 1.

Variables & Data Types

Assignment:

age = 13
price = 9.99
name = "Alice"
is_active = True

Data types:

intWhole numbers: 5, -3
floatDecimals: 3.14
strText: "hello"
boolTrue or False

Use snake_case for variable names: my_age, first_name

Input & Output 
print("Hello, World!")
name = input("Enter your name: ")
print("Hello,", name)

f-strings (format strings):

score = 95
print(f"Your score is {score}")

Concatenation:

greeting = "Hello, " + name + "!"
Type Casting 
age = int(input("Enter age: "))
price = float(input("Price: "))
text = str(42)   # "42"
count = int(3.7) # 3 (truncates)

Always cast input() to the correct type before arithmetic.

Comments 
# This is a single-line comment
x = 5  # inline comment
Operators

Arithmetic:

+Addition
-Subtraction
*Multiplication
/Division (float result)
//Integer (floor) division
%Modulus (remainder)
**Exponent / power

Comparison:

==Equal to
!=Not equal to
<Less than
>Greater than
<=Less than or equal
>=Greater than or equal

Logical:

andBoth conditions true
orEither condition true
notNegates condition

Assignment uses =. Equality test uses ==.

Selection — if / elif / else 
if score >= 90:
    print("A grade")
elif score >= 70:
    print("B grade")
else:
    print("Try again")

Always use 4 spaces for indentation. The colon : is required.

Iteration — for loop 
for i in range(5):
    print(i)       # 0 1 2 3 4

for i in range(1, 6):
    print(i)       # 1 2 3 4 5

for i in range(0, 10, 2):
    print(i)       # 0 2 4 6 8

Use for when the number of iterations is known.

Iteration — while loop 
count = 1
while count <= 5:
    print(count)
    count += 1

Use while when iterations depend on a condition.

Lists 
# Create
fruits = []               # empty list
colours = ["red", "blue"]

# Access (0-indexed)
print(colours[0])         # "red"
print(colours[-1])        # last: "blue"

# Modify
fruits.append("apple")
fruits.remove("apple")
del fruits[0]

# Loop
for fruit in fruits:
    print(fruit)

# Length
print(len(fruits))
Functions

Void (procedure):

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")

With return value:

def add(a, b):
    return a + b

result = add(3, 4)
print(result)  # 7

No parameters:

def print_header():
    print("=" * 20)
    print("CGA PyLab")
    print("=" * 20)

Function names use snake_case. Call before using if defined below.

Turtle Graphics 
import turtle

t = turtle.Turtle()
t.forward(100)
t.right(90)
t.forward(100)
t.color("blue")
t.pensize(3)

# Draw a square
for i in range(4):
    t.forward(100)
    t.right(90)
t.forward(n)Move forward n pixels
t.backward(n)Move backward n pixels
t.right(angle)Turn right by angle°
t.left(angle)Turn left by angle°
t.penup()Lift pen (no drawing)
t.pendown()Put pen down
t.color("red")Set pen colour
t.fillcolor("blue")Set fill colour
t.begin_fill()Start filled shape
t.end_fill()End filled shape
t.circle(r)Draw circle radius r
t.goto(x, y)Move to position
t.speed(n)Speed 0=fast, 1=slow
